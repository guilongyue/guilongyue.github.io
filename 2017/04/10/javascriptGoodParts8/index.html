javascript good parts 方法

发表于 2017-04-10 | 分类于 笔记 |

方法

Array

  1. array.concat(item…)
    concat方法返回一个新数组，它包含array的浅复制，并将1个或多个参数item附加在其后，如果参数item是一个数组，那么它的元素会被分别添加，此外参见arrary.push(item…);

  2. array.join(separator)
    join方法把数组变为字符串，用separator把他们连在一起，默认的方法separator是，。我们可以把空字符串作为separator。

  3. array.pop();
    push和pop使array像堆栈stack一样工作，pop移除array中最后一个元素
    并返回，如果array是空的他将返回undefined。

  4. array.push(item…)
    push方法将一个或多个参数item附加到一个数组的末尾，不像concat方法那样，他会修改该数组array，如果参数item是一个数组，他会将数组作为单个元素添加到数组中，它返回新array的长度。

  5. array.reverse();
    reverse方法反转array中的元素的顺序，他返回当前的array。

  6. array.shift()
    shift方法会移除数组array中的第一个元素并返回该元素，如果这个数组是空的，他会返回undefined。shift通常比pop慢的多；

  7. array.slice(start,end);
    slice方法对array中的一段做浅复制。第一个被复制的元素是array[start].他将一直复制到array[end]为止。end参数是可选的，被且默认值是该数组的长度array.length，若果两个参数中的任何一个是负数，array.length将和他们相加，视图使他们变成非负数。

  8. array.sort(comparefn)
    sort方法对array中的内容做适当的排序，他不能正确的给一组数字排序。
    js默认的比较函数假定所有要排序的元素都是字符串，没有检测他们的类型，幸运的是你可以用自己的比较函数来替换默认的比较函数，你的比较函数应该接受两个参数，并且若果两个参数相等则返回0，如果第一个参数要排在前面，则返回一个负数，如果第二个参数要排在前面则返回一个正数。

  9. array.splice(start,deleteCount,item…)
    splice方法从array中移除1个或多个元素，并用新的item替换他们，参数start是从数组array中移除元素的开始位置。参数deleteCount是要移除的元素合数。如果有额外的参数，那么item都将插入到所移除元素的位置上。他将返回一个被移除元素的数组。
    splice最主要的作用是从一个数组中删除元素，

  10. array.unshift(item…)
    unshift方法像push方法一样用于将元素添加到数组中，是吧item插入到开始部分而不是末尾。

Function

  1. function.apply(thisArg,argArray)
    apply方法调用函数function，传递一个将被绑到this上的对象和一个可选的参数数组。apply方法被用在apply调用模式中。

Number

  1. number.toExponential（fractionDigits）
    toExponential方法吧这个number转化成一个指数形式的字符串。
  2. number.toFixed(fractionDigits)
    toFixed方法把这个number转换为一个十进制形式的字符串。可选参数fractionDigits控制其小数点后的数字位数。0-20之间
  3. number.toPrecision(precision)
    把这个number转换成一个10进制形式的字符串，可选参数控制控制有效数字位数。0-21之间
    number.toString(radix);
    toString方法把这个number转换成字符串

Object

  1. object.hasOwnProperty(name)
    如果这个object包含了一个名为name的属性，那么hasOwnProperty方法返回true。原型链中同名属性是不会被检查的。这个方法对name就是hasOwnProperty时不起作用，此时返回false.

    RegExp

  2. regexp.exec(string);
    是最强大也是最慢的方法，如果他成功匹配regexp和string，他会返回一个数组，数组中下表为0的元素将包含正则表达式regexp匹配的子字符串，下标为1元素是分组一捕获的文本，下标为2的元素是分组2捕获的文本，以此类推，如果匹配失败返回null。
    如果regexp带有一个g标志(全局标志)，事情变得有点更加复杂了，查找不是从这个字符串的其实位置开始，而是从regexp.lastIndex（它初始化为0）位置开始。如果匹配成功，那么regexp。lastIndex将被设置为该匹配后第一个字符的位置。
  3. regexp.test（string）
    test方法是最简单和最快的方法，如果该regexp匹配该字符串，他会返回true，否则返回false。

String

  1. string.charAt(pos);
    charAt方法返回在string中pos位置处的字符。如果pos小于0，或大于等于length。他会返回空字符串，js没有字符类型，这个方法返回的结果是一个字符串。

  2. String.charCodeAt(pos)
    charCodeAt方法同charAt一样，只不过他返回的不是一个字符串，而是以整数形式表示的在String中的pos位置处的字符的字符码位，如果pos小于0或大于等于字符串的长度string.length,它返回NaN。

  3. string.concat(string…)

  4. string.indexOf(searchString,position);
    indexOf方法在string内查找另一个字符串searchString。如果找到，返回第一个匹配字符的位置，否则返回-1，可选参数可设置从string的某个位置开始查找。

  5. string.lastIndexOf(searchString,position)
    lastIndexOf方法和indexOf方法类似，只不过他是从字符串末尾开始查找。

  6. string.localeCompare(that)
    比较两个字符串，如果string比字符串thst小，那么结果为负数，相等结果为0.

  7. string.match(regexp)
    match方法匹配一个字符串和一个正则表达式。他根据g标识来决定如何进行匹配，如果没有g标识，那么调用string.match(regexp)的结果与regexp。exec(string)的结果相同。然而如果regexp带有g标识，那么他将返回一个包含除捕获分组之外的所有匹配的数组。
  8. replace方法string.replace（searchValue,replaceValue）
    replace方法对string进行查找和替换操作，并返回一个新的字符串。参数searchValue可以是字符串或正则表达式对象，如果他是一个字符串，那么searchvalue只会在第一次出现的地方替换。

如果searchValue是一个正则表达式且带有g标志。那么他将替换所有匹配之处，如果没有他将只替换第一个。
replaceValue可以是一个字符串或一个函数，如果replaceValue是一个字符串，字符$有特别的含义。还可以是函数，以后再说。

  1. string.search(regexp)
    search方法和indexOf方法类似，只是他接受一个正则表达式对象，而不是字符串，如果找到匹配，他返回第一个匹配的首字符位置，如果没有找到匹配，则返回-1.此方法会忽略g标志，且没有position参数。

  2. string.slice(start,end)
    slice方法复制string的一部分来构造一个新的字符串，如果start是负数，他将于string.length相加。end参数是可选的，且他的默认值是负数，那么他将与string。length相加。end参数，想要得到从位置p开始的n个字符，就用string。slice(p,p+n).此外。substring和array。slice。

  3. string.split(separator,limit)
    split方法就是把这个string分割成片段来创建一个字符串数组，可选参数，limit可以限制被分割的判断数量。separator可以是一个字符串或一个正则表达式。

  1. string.substring(start,end)
    和slice一样，只是他不处理负参数.

  2. string.toLowerCase(); toUpperCase();
    返回一个新的字符串。

  3. string.fromCharCode(char…)
    从一串数字返回一个字符串。
