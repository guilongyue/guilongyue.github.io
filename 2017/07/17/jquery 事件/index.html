JQuery 简述

on方法用来绑定一个事件，需要传入事件名称和对应的处理函数。

off移除绑定

插件的编写

本质上JQuery插件是定义在Jquery构造函数的prototype对象上的方法，这样所有势力节能共享这个方法，因为Jquery构造函数的prototype对象被简写为JQuery.fn

常用的工具方法

  • $.trim()
  • $.contains
  • $.each
    Jquey实例也有一个each方法，两者作用差不多
  • $.map()
    遍历数组和对象，但是会返回一个新对象。
  • $.inArray
  • $.extend用于将多个对象合并进第一个对象
  • 另一种用法是生成一个新对象，用来继承原对象，他的第一个参数应该是空对象。
  • $.proxy 方法类似于bind方法，可以绑定函数的上下文对象和参数，返回一个新函数。
  • $.data $.removeData
  • $.parseHTML方法将字符串解析为DOM对象
  • $.parseJSON 解析为js对象，作用类似JSON.parse()
  • $.parseXML
  • $.makeArray转换为数组。
  • $.merge() 合并数组
  • $.now() 毫秒数

    判断类型

    isArray（） isEmptyObject（）
    isFunction() isNumberic()
    isPlainObject()
    isWindow() isXMLDoc()

    Ajax

    accepts: 本机所能处理的数据类型，告诉服务器；
    async 默认为true 异步的；
    complete 请求结束时执行的回调函数。
    contentType 发送到服务器的数据类型
    data 向服务器发送的数据。
    error 请求失败时的回调函数。
    jsonp
    success 请求成功时的回调函数。
    timeout，等待的最长毫秒数。
    datatype 向服务器请求的数据类型。text html json jsonp xml

    渐变写法

    $.get() $.getScript()读取一个脚本并执行
    $.getJSON 读取一个JSON文件
    $.post（）
    $.fn.load()； 是实例上的方法。

    Ajax事件

    ajaxComplete（） ajax请求完成
    ajaxError（） ajax请求出错
    ajaxSend（） ajax请求发出之前
    ajaxStart
    ajaxStop 所有ajax请求完成之后
    ajaxSuccess()请求成功之后。
