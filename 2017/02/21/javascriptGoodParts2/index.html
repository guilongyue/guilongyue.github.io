javascript good parts 语法

发表于 2017-02-21 | 分类于 笔记 |

语法

标识符

JavaScript 不允许使用保留字作为参数和变量名。也不允许用在对象字面量中。

数字

内部表示为64为的浮点数，他没有分离出整数类型，所以1.0和1是相同值。

负数可以用前缀运算符来表示。 - ；
值NaN是一个数值，他表示一个不能产生正确结果的运算结果，NaN 不等于任何值，包括他自己。可以用isNaN（number）检测NaN。
值Infinity表示所有大于1.797 e+308的值。
数字拥有方法，Math.floor(number) 将一个数字转换为整数。

字符串

字符串字面量可以被包含在单引号或双引号中。包含0或多个字符。\是转义字符。js中所有字符都是16位的。
转义字符允许吧正常情况下不被允许的字符插入到字符串中，包括\,引号，控制字符。\u 约定允许指定用数字表示的字符码位。

“A” === “\u0041”

字符串有一个length属性，表示字符串的长度。
字符串是不可变的，一旦字符串被创建就永远无法改变它，但可以用 + 链接其他字符串得到新字符串。

‘c’ + ‘a’ + ‘t’ === ‘cat’ 是true；
‘cat’.toUpperCase() === ‘CAT’

语句

在web浏览器中，每个script标签，都提供一个被编译且立即执行的编译单元，js把他们一起抛入一个全局名字空间。
swith,while ,for do 允许有一个可选的前置标签label，他配合break语句来使用。
语句往往按照从上到下的顺序被执行，js可以通过条件语句(if,switch),循环语句（for，while，do），强制跳转语句（break，return，throw）和函数来改变执行序列。
代码块是包含在花括号内的一组语句，js代码块不会创建一个新的作用域，因此变量应定义在函数的顶部，而不是代码块内。
if语句根据表达式的值来改变程序执行流程。
下列值为假：false，’’,undefined,null,NaN,0
其他所有的值被当做true。
switch语句执行一个多路分支。它把表达式的值和所有指定的case条件进行匹配，其表达式可能产生一个数字或字符串，当找到一个精确匹配是，执行case从句中的语句。当没有找到任何匹配值，执行可选的default语句。
case表达式不一定必须是常量，为防止继续执行下一个case，case语句有应该跟一个break语句退出switch语句。
while语句执行一个简单的循环。
for语句是一个结构更复杂的循环语句。他有两种形式。
常见的形式是三个可选从句控制，初始化语句(initialization),条件从句（condition）和增量从句(increment)。首先initialization被执行，它的作用是初始化循环变量，接着计算condition的值，典型情况根据一个完成条件检测循环变量，如果condition省略，假定返回的值为真，如果condition的值为假，循环终止。否则执行代码块，执行increment。接着循环会重复执行condition。
另一种法师被称为for in语句，会枚举一个对象的所有属性名（键名），在每次循环中，对象的另一个属性名字符串被赋值给for in之间的变量
通常需要检测object.hasOwnProperty(variable) 来确定这个属性名是该对象的成员，还是从原型链里继承到的。

do语句就像while语句，唯一区别是她在代码块执行之后而非执行前检测表达式的值，这就意味着代码块至少执行一次。
try 语句执行一个代码块，并捕获该代码块抛出的任何异常。catch从句定义了一个新的变量，他将接受该异常对象。
throw语句抛出一个异常，如其在一个try代码块中，那么控制权会调到catch从句中，如果throw语句在函数中，则该函数调用被放弃，且控制权会跳到调用该函数的try语句的catch从句中。
throw 语句中表达式的值通常是一个对象字面量，它包含一个name属性和messgae属性，异常捕获器可以使用这些信息决定去做什么。
return语句会使一个函数提前返回，它可以指定要返回的值，若没有指定返回表达式，那么其返回值为undefined。
js 不允许在return和表达式之间换行。
break语句用于退出循环语句和我、switch语句，他可以指定一个可选的标签。那将会使程序退出带该标签的语句。
js不允许在break和标签之间换行。
一个expresssion语句可以给一个或多个变量或成员赋值，或者调用一个方法，删除一个属性，运算符 = 被用于赋值，不要和 === 混淆，运算符 += 用于连接字符串、加法运算。

表达式

最简单的表达式是字面量值(字符串和数字)，变量、内置的值(true，false，undefined、null、NaN和Infinity)，以new前导的调用表达式，以delete前导的属性存取表达式，包在圆括号中的表达式，以一个前缀运算符作为前导的表达式，或者表达式后面跟着：

  • 一个插入运算符与另一个表达式。
  • 三元运算符？后面跟着一个表达式，然后接着：然后再接第三个表达式。
  • 一个函数调用
  • 一个属性存取表达式

三元运算符有三个运算数，如果第一个运算数为真，他产生第二个运算数的值，但是，如第一个运算数为假，它会产生第三个运算数的值。
运算符优先级越高，结合性越强。括号可以改变运算符的优先级。

运算符 解释
.[]() 属性存取函数调用
delete new typeof + - ! 一元运算符
* \ % 乘除取模
+ - 加法链接减法
>= <= > < 不等于运算符
=== !== 等式运算符
&& 逻辑与
\ 逻辑或
?: 三元

typeof 运算符产生的值有 ‘number’,’string’,’object’,’undefined’,’boolean’,’function’, 数组和null 是’object‘。这是不对的。
如果！的运算数的值为真，那么产生false，否则产生true。
+运算符可以进行字符串连接或加法运算，如果要进行加法运算，确保两个运算数都是数字。
/运算符可能会产生小数，即使两个运算数都为整数。
&& 如果第一个运算的值为真，那么产生第二个运算数的值。
|| 如果第一个运算数的值为真，那么他产生第一运算数的值。否则他产生第二个运算数的值。
函数调用引发函数的执行，函数调用运算符是跟在函数名后面的一对圆括号，圆括号中可能包含将会传递给这个函数的参数。
一个属性存取表达式，用户指定一个对象或数组的属性或元素。

字面量

对象字面量是一种方便指定新对象的表示法，属性名可以使标识符或字符串，这些名字被当做字面量名而不是变量名对待。所以对象的属性名才能知道，属性的值就是表达式。
数组字面量是一种表示新数组的表示法。

函数


函数字面量定义了函数值，它可以有一个可选的名字，用于递归的调用自己，它可以指定一个参数列表，这些参数将作为标量，由调用时传递的实际参数初始化，函数的主题包括变量定义和语句。
